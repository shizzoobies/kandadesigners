Web Design & API Integration

FixAlways

A full-stack Florida home services directory powered by real vendor data and an AI chat assistant.

Google Places API

Vendor data across 188+ Florida cities is pulled and structured using the Google Places API. Each listing is automatically enriched with business name, address, phone, ratings, and review counts — building a live, searchable database without manual data entry.

Anthropic Claude API

An embedded chat assistant powered by Claude's API lets users describe their home service problem in plain language and get matched to relevant service categories and local providers. Context-aware, fast, and built entirely through the Anthropic API.
